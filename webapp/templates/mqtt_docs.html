MQTT Documentation

Topics Overview
Outbound (ESP32 → Server)
  • games/chess/start - Start a new game
  • games/chess/move - Make a move
  • games/chess/forfeit - Forfeit game
  • games/chess/remis - Request/accept draw
  • games/chess/timeout - Time ran out
  • games/chess/cancel - Cancel game
Inbound (Server → ESP32)
  • games/chess/response/<board_id>/<game_id> - Server responses
Message Formats
Start Game
{
    "board_id": "Board-001",
    "rfid_white": "abc123",
    "rfid_black": "xyz789",
    "mode": "blitz",
    "time_limit": 300
}
Make Move
{
    "board_id": "Board-001",
    "game_id": "123",
    "rfid": "abc123",
    "move": "e2e4"
}
Server Response
{
    "status": "legal"
}
// or
{
    "status": "checkmate",
    "winner": "abc123"
}
